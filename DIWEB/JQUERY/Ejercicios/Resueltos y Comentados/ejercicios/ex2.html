¿Qué solución aporta este ejercicio?

Al colocar el cursor sobre una etiqueta <h1> con la clase .camaleon, cambia de color.
Recupera el color previo al mover el ratón fuera del elemento.

Puedo cambiar de color

<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js"></script>
<script type="text/javascript">
$(function(){
//Seleccionamos las etiquetas h1 con la clase camaleon con $("h1.camaleon")
//El evento .mouseover() se lanza cuando el raton pasa por encima del elemento.
   $("h1.camaleon").mouseover(colorWhite);
//El evento .mouseout() se lanza cuando el raton deja de estar encima del elemento.
   $("h1.camaleon").mouseout(colorBlack);
});
function colorWhite(){
//usar $(this) nos permite seleccionar la etiqueta que llamó a este método a través del evento.
//Cambiamos su propiedad css, por parámetros le indicamos que estilo queremos cambiar y que valor le queremos dar a ese estilo.
   $(this).css("color","rgba(255,255,255,1.00)")
}
function colorBlack(){
//funciona de forma similar que el método colorWhite().
   $(this).css("color","rgba(0,0,0,1.00)")
}
</script>
